

First stab at a utility to review a document and identify all versions of person names. 

During the creation and editing of a clinical report about a person, repetitive text may be included from a similar study and tweaked. Ocassionaly a name is missed and there will be one reference to a "Sue" in a report for Freddie. 

Should probably be a plugin to LibreOffice or MSword. 


Open file (try to handle any word processing file)

Find all words.
Sort
lowercase
de-dupe
remove dictionary words
remove numbers
remove place names
remove custom dictionary words (need custom dict interface)
